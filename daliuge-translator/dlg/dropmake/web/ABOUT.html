The DALiuGE Translator is a component of the DALiuGE system. It takes input from the EAGLE workflow graph editor and transforms that into a directed acyclic graph (DAG) for execution in the DALiuGE execution engine. In addition to just translating the graph, it also contains functionality to partition and optimally statically schedule the DAG for the deployment onto a compute cluster. More information can be found in the translator section of the DALiuGE documentation.
